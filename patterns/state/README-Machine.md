本文主要介绍「有限状态机」如何在真实业务场景中使用。

「有限状态机」比较简单，就是算法的选取取决于于自己的内部状态。相较于「策略模式」算法的选取由用户决策变成内部状态决策，「策略模式」是用户(客户端)选择具体的算法，「有限状态机」只是通过内部不同的状态选择具体的算法。
